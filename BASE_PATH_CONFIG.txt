GitHub Pages Base Path Configuration
====================================

Your site is deployed at: https://tomergodelli.github.io/Transpose/
                                                        ^^^^^^^^^^^
                                                        This is the base path!

IMPORTANT: The repository name "Transpose" becomes part of the URL.

How it works now:
-----------------

1. <base href="/Transpose/"> in index.html
   - Makes all relative URLs work correctly
   - player.js → /Transpose/player.js
   - public/tracks.manifest.json → /Transpose/public/tracks.manifest.json

2. URL Routing:
   - Home: https://tomergodelli.github.io/Transpose/
   - Collection: https://tomergodelli.github.io/Transpose/hof
   - Song: https://tomergodelli.github.io/Transpose/gavriel

3. Path Processing:
   - JavaScript strips "/Transpose/" from pathname
   - Then processes the route (e.g., "hof", "gavriel")

If you change the repository name or use a custom domain:
----------------------------------------------------------

Update these files:
1. index.html: <base href="/NEW_REPO_NAME/">
2. index.html: .replace(/^\/NEW_REPO_NAME\/?/i, '')
3. 404.html: window.location.replace('/NEW_REPO_NAME/');

For custom domain (e.g., music.example.com):
Use: <base href="/">
And remove the pathname replacement logic.
